# GitHub Actions Matrix Build Demo

This repository demonstrates a GitHub Actions workflow with matrix builds and artifact management.

## Workflow Features

- ✅ Matrix build with 6 different variants
- ✅ Parallel execution across multiple OS and Node.js versions
- ✅ Artifact upload with prefix `build-4eee635-`
- ✅ Required identifier: `matrix-4eee635`
- ✅ Non-empty build artifacts

## Matrix Variants

- Ubuntu + Node.js 18
- Ubuntu + Node.js 20  
- Ubuntu + Node.js 21
- macOS + Node.js 18
- Windows + Node.js 20
- Windows + Node.js 21
